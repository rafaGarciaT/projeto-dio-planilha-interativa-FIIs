# projeto-dio-planilha-interativa-FIIs

Este é um desafio de projeto que faz parte do curso da Dio "Santander - Excel com Inteligência Artificial." Aqui, desenvolvi uma planilha que simula investimentos em Fundos Imobiliários, e serve como um guia para onde direcionar novos investimentos.

## Principais funcionalidades
- **Configuração do Investimento**:
Permite definir salário, percentual e valor investido mensalmente, horizonte de tempo, taxa de rendimento e perfil de risco, que servem de base para todos os cálculos.

- **Distribuição Sugerida por Perfil**:
Apresenta a alocação ideal entre os diferentes tipos de FIIs conforme o perfil selecionado, comparando a distribuição sugerida com a atual e destacando desvios percentuais.

- **Simulação de Resultados**:
Calcula o patrimônio acumulado ao final do período definido e estima o valor mensal de dividendos gerados pela carteira.

- **Projeções de Longo Prazo**:
Mostra projeções de patrimônio e dividendos para diferentes horizontes de tempo (2, 5, 10, 20 e 30 anos), facilitando a visualização do efeito dos aportes recorrentes e dos juros compostos.

- **Registro de Fundos**:
Espaço destinado ao cadastro dos FIIs da carteira, permitindo acompanhar valores investidos por tipo de fundo.
